# Test Audio for Rakarrack

This folder contains dry guitar recordings intended for effect-testing in Rakarrack.

## Purpose

These clips are provided as repeatable input material so effect behavior can be tested and compared consistently without requiring a live guitar or audio interface input.

Typical workflow:

1. Run `rakplay`.
2. Load one of the WAV files from this folder.
3. Connect `rakplay` JACK outputs to Rakarrack inputs.
4. Toggle effects and compare results using the same source performance.
